# CoffeeMan Travel Link Saver - Trip Link Organizer mini app

A simple and beautiful web app designed to help you organize and save travel links and videos from multiple social media platforms. Built with HTML, Tailwind CSS, and JavaScript.

## 🚀 Features

✅ Create and manage multiple travel destinations/trips.  
✅ Save links from major platforms: YouTube, TikTok, Instagram and Xiaohongshu (XHS).  
✅ Preview saved videos (where embedding is supported).  
✅ Smart link fallback provides a direct link for restricted content
✅ Stored locally in your browser (no account needed).  

## 📱 How to Use

Click the + button and select "New Destination" (My Trips view).

Give your trip a name (e.g., "Kyoto Food Guide").

Click on the destination name to view saved posts for that trip.

Click the + button again and select "Save Content" (inside a trip).

Paste a link or share text into the box. The URL will be automatically extracted. Add an optional note.

Click a saved item to open the built-in content viewer.

Use the "Back" button to return to the list of trips.

## 🔐 Data Storage & Privacy

All data is saved locally on your device using Local Storage.
This means your data stays private and is not uploaded anywhere.
If you clear browser cache or use another device, your data will be lost.

# CoffeeMan旅行链接收藏夹 迷你小程序

一个简单又美观的网页应用，帮你轻松整理和收藏来自多个社交媒体平台的旅行链接和视频。 基于 HTML、Tailwind CSS 和 JavaScript 开发。

## 🚀 功能特点

✅ 创建和管理多个旅行目的地/行程。  
✅ 收藏来自主流平台（YouTube、TikTok、Instagram 和小红书）的链接到指定行程中。  
✅ 直接在应用内预览支持内嵌的内容（如 YouTube）。  
✅ 智能链接回退：对于限制播放的视频，提供直接在原生应用中打开的链接。  
✅ 数据保存在你的浏览器本地（无需注册账号）。  

## 📱 使用方法

在“我的行程”页面，点击 + 按钮，选择“New Destination”（新目的地）。

输入行程名称（例如：“京都美食攻略”）。

点击行程名称，进入该行程的收藏列表。

再次点击 + 按钮，选择“Save Content”（保存内容）。

粘贴链接或分享文本（URL 会自动提取）。可添加可选备注。

点击已保存的条目，打开内容预览器。

使用“Back”按钮返回行程列表。

## 🔐 数据存储与隐私

所有数据都保存在你设备的本地存储（Local Storage）。
你的数据不会上传到任何地方，完全私密。
如果你清空浏览器缓存或换设备，数据会丢失。
